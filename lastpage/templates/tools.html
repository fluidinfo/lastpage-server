Privacy: You only publish URLs that you explicitly choose, by clicking the ﬁ icon. Only one URL is stored, so clicking on a new page clears the previous value. You can clear the record of the single stored page by choosing "Clear" from the right-click menu of the ﬁ icon. Your current lastpage.me page can be seen by anyone.

Tools

  • We have a command line script called follow. Just type follow username and your browser opens on the page they're looking at.
  • We also have a command line script called imat. Type imat http://whatever.com and it stores the given URL as your current page.
