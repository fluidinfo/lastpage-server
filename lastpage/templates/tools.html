lastpage.me

There's lots more to know about lastpage.me

Privacy

  • You only publish the URL you explicitly choose by clicking
  • Only one URL is stored at a time; clicking on any page overwrites the old URL
  • Your current lastpage.me page can be seen by anyone

Advanced mode & custom URLs

  • The Chrome extension for lastpage.me has an advanced mode
  • Enable it in options (right-click )
  • Click to add a page, then choose either "Save" (for the regular behavior) or "Custom" to enter a URL suffix. E.g., enter "xxx" and the URL http://lastpage.me/yourname/xxx will send visitors to your current page
  • Use advanced mode to create as many custom URLs as you like

lastpage.me is open in many ways

  • Get the source of the lastpage.me server
  • Get the source of the Chrome extension
  • We've released a command-line script for setting/getting/opening URLs
  • URLs are stored in Fluidinfo, which means lastpage.me has an API
  • You own your own data in Fluidinfo, so you can write applications to work with it, or use those written by others
  • Fluidinfo even lets you change permissions on your underlying tags
