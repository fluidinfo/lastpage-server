lastpage.me

Your privacy, custom URLs, open source, Fluidinfo, command-line tools, feedback.

Your privacy

  • You only publish the URL you explicitly choose by clicking .
  • Only one URL is stored at a time; clicking on any page overwrites the old URL.
  • Your current lastpage.me page can be seen by anyone.

Custom URLs

  • The Chrome extension has an advanced mode.
  • Enable it in options (right-click on ).
  • When adding a page choose either "Save" (for the regular behavior) or "Custom" to enter a URL suffix. E.g., enter "xxx" and the URL http://lastpage.me/yourname/xxx will send visitors to your current page.
  • Use advanced mode to create as many custom URLs as you like.

lastpage.me is open source

  • Get the source of the lastpage.me server.
  • Get the source of the Chrome extension .
  • We've released a command-line script for setting/getting/opening URLs.

lastpage.me uses Fluidinfo, so its data is open too

  • URLs are stored in Fluidinfo, which means 3rd party applications can work with it.
  • In other words, lastpage.me data has a full API.
  • You control your own data in Fluidinfo: you can add lastpage.me compatible tags any way you like, or use any application to access them.
  • Fluidinfo even lets you change permissions on your underlying tags.
